# Security Policy

## Scope

caddy-ui is designed for **LAN/trusted-network use only**. It is not intended to be exposed to the public internet, though there is no inherent risk in its design, its it always best to be very safe with someone as consequential as control of a whole webserver/proxy. Please read the threat model in the README before deploying.

## Reporting a Vulnerability

If you discover a security vulnerability, **open a public GitHub issue.**

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

## Supported Versions

Only the latest commit on `main` is actively supported.
